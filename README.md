# learn-vue3-20260106


Vue3学習用

https://ja.vuejs.org/tutorial/

チュートリアルを若干アレンジして実装

npm/voltaなどをインストールしておく

```shell
# clone
$ git clonse https://github.com/rx21js6/learn-vue3-20260106.git

# 移動
$ cd learn-vue3-20260106

# 依存ライブラリのinstall
$ npm install

# 実行（実API）
$ npm run dev

# 実行（Mock Service Worker）
$ npm run dev:mock

# http://localhost:5173
# にアクセスする。

# Vitest実行
$ npm run test
````
